Sophon Inference User Manual¶

Tips

Before using “Sophon Inference”, make sure you have installed the Bitmain Sophon-SC3 TPU and the software development kit BMNNSDK; And read the “NNToolChain User Documentation” and “BMCV User Documentation.”
